Events

A event is a particular activity like a conference or a movie (which may happen at one or more places).

As a user, I can search for events by type or tag.

As a user, I can view reviews and places that are associated with an event.

As a user, I can review an event if I have a tag that matches the event. (for example to review a movie you must have a tag for that movie)
